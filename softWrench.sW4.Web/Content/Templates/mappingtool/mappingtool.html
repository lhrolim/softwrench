Syntax Help:

  • Left column represents the source values, the conditions
  • Right column represents the values that would be applied if the condition(s) match
  • Comma Sepparated values --> Could be used on either the source or the destination column. Multiple value matching (ex: x,y applies x and y if a condition is matched, or if any of the conditions match apply a value)
  • And Source mathing --> Both conditions need to exist (ex: x&y applies a mapping only if both conditions x and y are matched)
  • a @default source/condition could be set --> these values would be applied everytime regardless of any sort of matching (ex: a default profile could be set to all users like that)
